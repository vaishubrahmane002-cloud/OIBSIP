# 🐻 Super Kawaii To-Do List

## 📌 Project Overview
A beautiful, aesthetically pleasing, and highly functional To-Do List application. Built as part of the **Oasis Infobyte Web Development Internship (OIBSIP)**, this tool helps users track and manage their daily tasks efficiently while providing a delightful user experience.

## ✨ Features
- **Add, Edit, & Delete:** Full control over your task list.
- **Task Segregation:** Automatically separates "Pending ⏳" and "Completed ✅" tasks.
- **Time Tracking:** Records and displays the exact date and time when a task was added.
- **Dynamic Progress Bar:** Visually tracks your productivity by showing the percentage of completed tasks.
- **Dark/Light Mode Toggle:** Switch seamlessly between a soothing light theme and a sleek dark theme.
- **Data Persistence:** Uses browser **LocalStorage** so your tasks are saved even if you refresh or close the page.
- **Scrollable UI:** Fixed container layout with custom scrollbars to maintain a clean design even with multiple tasks.

## 🛠️ Technologies Used
- **HTML5:** For structuring the application.
- **CSS3:** For custom styling, floating background animations, glassmorphism effects, and responsive flexbox layouts.
- **Vanilla JavaScript:** For core logic, DOM manipulation, event handling, and LocalStorage integration.

## 🚀 How to Run the Project
1. Open the project folder.
2. Ensure `index.html`, `style.css`, `script.js`, and the profile image (`img 2.png`) are in the same directory.
3. Simply double-click the `index.html` file to open it in your default web browser.
