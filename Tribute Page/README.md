# 🚀 Kalpana Chawla Tribute Page

## 📌 Project Overview
A beautifully designed, immersive tribute page dedicated to **Kalpana Chawla**, the first woman of Indian origin to travel to space. This project was developed as part of the **Oasis Infobyte Web Development Internship (OIBSIP)**. It goes beyond a simple static page by integrating a modern UI and dynamic background animations to match the aerospace theme.

## ✨ Features
- **Dynamic Starry Background:** Uses HTML5 `<canvas>` and JavaScript to generate a custom, continuous falling star animation.
- **Glassmorphism UI:** Features translucent, blurred cards with glowing cyan hover effects for a modern, sci-fi aesthetic.
- **Floating Animations:** The hero portrait has a smooth, CSS-based continuous floating animation to simulate zero gravity.
- **Responsive Layout:** Built with CSS Grid and Media Queries to ensure a seamless experience across mobile, tablet, and desktop devices.
- **Custom Theming:** Includes a custom-styled scrollbar, Orbitron typography, and text-stroke effects to complete the space vibe.

## 🛠️ Technologies Used
- **HTML5:** Semantic structuring of the content, timeline, and achievements.
- **CSS3:** Advanced styling including backdrop-filters, keyframe animations, glow effects, and responsive design.
- **Vanilla JavaScript:** For the Canvas API logic to draw, animate, and continuously update the falling stars.

## 🚀 How to Run the Project
1. Open the project folder.
2. Ensure you have the `Img` folder containing `kalpana img.jpg` and `back.jpg`.
3. Keep `index.html`, `style.css`, and `script.js` in the same directory.
4. Simply double-click the `index.html` file to launch the webpage in your browser.
