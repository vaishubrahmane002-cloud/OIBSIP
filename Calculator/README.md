# 🎀 Cute Aesthetic Calculator

## 📌 Project Overview
A fully functional, visually appealing calculator built as part of the **Oasis Infobyte Web Development Internship (OIBSIP)**. This project goes beyond basic arithmetic by offering advanced mathematical operations and complete keyboard accessibility, all wrapped in a delightful pastel user interface.

## ✨ Features
- **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
- **Advanced Operations:** - `x²` (Square)
  - `x³` (Cube)
  - `xʸ` (Power)
  - `π` (Pi value)
- **Full Keyboard Support:** You can use your keyboard to calculate! 
  - Numbers and basic operators work natively.
  - `Enter` or `=` for result.
  - `Backspace` to delete the last entry.
  - `Escape` or `c` to clear (AC).
  - Special hotkeys: `s` (Square), `b` (Cube), `^` (Power), `p` (Pi).
- **Interactive UI:** Smooth button press animations (both mouse clicks and keyboard presses), hover effects, and a responsive layout.
- **Error Handling:** Safe mathematical evaluations using try-catch blocks to prevent crashes on invalid inputs.

## 🛠️ Technologies Used
- **HTML5:** For the calculator's structure and layout.
- **CSS3:** For the pastel color palette, grid layout, shadows, and interactive keypress animations.
- **Vanilla JavaScript:** For DOM manipulation, mathematical logic, and complex event listeners (keyup/keydown).

## 🚀 How to Run the Project
1. Open the project folder.
2. Ensure `index.html`, `style.css`, `script.js`, and the decorative image (`bow.png`) are in the same directory.
3. Simply double-click the `index.html` file to open it in your default web browser.
