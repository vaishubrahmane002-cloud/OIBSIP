# 🚀 Oasis Infobyte Web Development Internship (OIBSIP)

Welcome to my repository! This contains the tasks I successfully completed during my **Web Development and Design Internship** at **Oasis Infobyte**. 

Throughout this internship, I built interactive, responsive, and functional web applications using frontend technologies and backend server logic.

---

## 📂 Projects Included

### 1. 🎀 Cute Aesthetic Calculator (Task 1)
A fully functional calculator that goes beyond basic arithmetic. It features a delightful pastel UI and complete keyboard accessibility.
* **Key Features:** Advanced operations (Square, Cube, Power, Pi), comprehensive keyboard event binding, and interactive press animations.
* **Tech Stack:** HTML5, CSS3, Vanilla JavaScript.

### 2. 🌌 Kalpana Chawla Tribute Page (Task 2)
An immersive, space-themed tribute page dedicated to Kalpana Chawla.
* **Key Features:** Continuous falling stars animation using HTML Canvas, glassmorphism UI cards, floating hover animations, and a fully responsive layout.
* **Tech Stack:** HTML5, CSS3, JavaScript (Canvas API).

### 3. 🐻 Super Kawaii To-Do List (Task 3)
A highly productive task management tool with an adorable, user-friendly interface.
* **Key Features:** Dark/Light mode toggle, dynamic progress bar, timestamps for tasks, and data persistence so tasks are saved automatically.
* **Tech Stack:** HTML5, CSS3, JavaScript (LocalStorage).

### 4. 🐾 Interactive Full-Stack Login Authentication (Task 4)
A secure login, registration, and dashboard system with a highly engaging animated SVG bear that reacts to user input.
* **Key Features:** Bear hides eyes when typing a password, secure routing, local JSON database for storing user credentials, and dynamic user greeting.
* **Tech Stack:** HTML5, CSS3, JavaScript, Node.js, Express.js.

---

## 🛠️ Overall Technologies Used
* **Frontend:** HTML, CSS (Flexbox/Grid, Animations), JavaScript (DOM Manipulation, Canvas).
* **Backend:** Node.js, Express.js.
* **Database:** Local JSON (File System).
